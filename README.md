# Afshaan & Laiba — Together RPG

A full static Phaser RPG version of the five-month anniversary game.

## Play

- GitHub Pages: https://afshaankhan.github.io/afshaan-laiba-phaser-rpg/
- Original Zo version: https://epicgem.zo.space/5months

## Controls

- Move: Arrow keys / WASD
- Interact and continue: E, Enter, or Space
- Settings / pause: O or Esc during gameplay
- Character customization: C from the title screen
- Journal: J
- Manual save: S
- Music: M
- Mobile: on-screen D-pad and action buttons

## Features

- Phaser 3 game engine
- Title menu, settings menu, pause/resume, and save controls
- Character selection and customization for Afshaan and Laiba
- Grounded 16-bit walking sprites with directional walk frames
- Partner character follows as the opposite person
- RPG-style tile maps, props, portals, collision, quests, and dialogue
- Together World overworld with growing places, errands, and memory gates
- Memory-shard collection in every chapter before the next portal unlocks
- Journal screen with chapter progress, play time, and collected shards
- Autosave, manual save, and resume progress with `localStorage`
- Settings and customization persistence with `localStorage`
- Mobile-friendly controls
- Final anniversary garden with a live timer since January 9, 2026 at 4:00 PM
- Gentle Forest 16-bit forest/map art integrated into the overworld and memory scenes
- Real Mana Seed Character Base spritesheets for Afshaan and Laiba walking animation
- Free cat companion/object art in Together World

This is a static browser game with no build step.
